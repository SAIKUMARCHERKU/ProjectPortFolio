Data Cleaning on World layoffs using MySQL

This project involved meticulous data cleaning and standardization of the World_layoffs dataset using advanced SQL techniques. Key steps included creating a staging table, removing duplicates with window functions, standardizing data entries, handling null values with self-joins, and formatting date columns. These operations ensured data consistency and integrity. This work demonstrated high SQL proficiency and the ability to prepare complex datasets for accurate analysis.

Case Study

Exploratory Data Analysis on World layoffs using MySQL

This project involved a comprehensive Exploratory Data Analysis (EDA) of the World_layoffs dataset using advanced SQL techniques. Initial data exploration was followed by aggregations to uncover key statistics and trends, such as major layoff events and industry-specific impacts. Common Table Expressions (CTEs) were utilized for advanced analysis, including yearly layoffs by company and rolling totals. This EDA provided significant insights, demonstrating my SQL proficiency and ability to extract actionable intelligence from complex data.
